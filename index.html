Piano Accompanist Web (MVP)

목표: 1~다수 악보 이미지를 OMR로 인식해서 기타프로처럼 반주를 자동 연주하는 웹앱. 지금은 첫 단계로, MusicXML/MIDI 파일을 직접 업로드해서 피아노 반주를 재생/표시하는 버전입니다.

1. MusicXML 업로드 → 악보 표시 (연주 준비)

먼저 외부 OMR/기보 프로그램(예: QuickStave, MuseScore 등)에서 악보 이미지를 MusicXML로 변환한 뒤, 여기에서 MusicXML 파일을 업로드하면 브라우저에서 악보를 렌더링합니다.

2. 피아노 반주 MIDI 업로드 → 브라우저에서 재생

피아노 반주만 들어있는 MIDI를 업로드하면 바로 브라우저에서 재생할 수 있습니다. (향후에는 위 MusicXML에서 피아노 파트만 추출해서 자동으로 이 영역으로 넘길 예정입니다.)
